![CirrusPad Banner](assets/banner.png)

## Architecture

CirrusPad is built using **C++** and **Qt 6**, leveraging the power of **QML** for its dynamic and responsive user interface. The backend uses a custom hierarchical file system model to handle notes, todos, and folders efficiently.
